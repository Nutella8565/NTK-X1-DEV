# 🚀 NTK-X1-DEV

**Developer Edition** (This repository is intended **for developers only.)


## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Nutella8565/NTK-X1-DEV.git

Go into the project folder:

```bash
cd NTK-X1-DEV
```

Install dependencies:

```bash
npm install
```

Start the application:

```bash
npx electron nutella-toolkit.js
```

---

## 🔒 Build

Create the encrypted version:

```bash
npm run build-secure
```

Build the Windows executable:

```bash
npm run build-win
```
